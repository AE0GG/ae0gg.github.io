# MeshCore
Google's AI Overview
> MeshCore is an open-source, decentralized off-grid communication system that enables secure, encrypted text messaging using LoRa radio hardware without relying on cellular or internet networks. It acts as a lightweight, portable C++ library designed for mesh networking, allowing devices to act as nodes, repeaters, or sensors to extend communication range.
